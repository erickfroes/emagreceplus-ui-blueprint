## Fluxos integrados principais

### Lead para paciente pagante
Lead → avaliação → pacote → contrato → assinatura → cobrança → paciente ativo → app liberado.

### Atendimento completo
Agenda → fila → atendimento → SOAP/anamnese → prescrição/plano → documento → assinatura → evidência → retorno.

### Pacote e financeiro
Pacote vendido → parcelas → contas a receber → pagamento → caixa → recibo → inadimplência/repasses/relatórios.

### Estoque operacional
Compra → entrada/lote/validade → atendimento/venda/saída → baixo estoque → pedido de compra → relatório.

### Documento assinado
Documento → D4Sign dispatch → webhook → validação → evidência → pacote JSON → auditoria → notificação.
